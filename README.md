Student Attendance Management System
1. Introduction
This system helps teachers manage student attendance, track topics taught, and enforce exam
eligibility rules based on attendance percentage.
2. Objectives
• Record daily attendance • Track topics taught • Calculate attendance percentage • Enforce 75%
eligibility rule • Generate weekly and semester reports
3. Features
• Student Management • Daily Attendance Recording (Date + Topic) • Automatic Attendance
Calculation • Exam Eligibility Check • Weekly Report Generation • Semester Final Report Export to
Excel
4. Attendance Formula
Attendance % = (Days Present / Total Classes) × 100
5. Eligibility Rule
Students with attendance below 75% are not allowed to sit for exams.
6. Weekly Report
Includes student attendance summary and topics covered each week.
7. Final Semester Report
Includes total classes, attendance percentage, and eligibility status for each student.

8. Technologies
Python, Tkinter, SQLite/MySQL, openpyxl or pandas for Excel export.
9. Conclusion
The system improves attendance tracking, ensures fairness in exam eligibility, and automates
reporting.
